# BATSEL-Bike-Anti-theft-System-with-Engine-Locking-
BATSEL (Bike Anti-Theft System with Engine Locking) is an ESP32-based motorcycle security system that disables the ignition coil using a relay to prevent unauthorized starting. It uses RF communication for internet-free control and includes MPU6050 motion detection, alarm alerts, horn/headlight control, and remote self-start.
